# Using Git Worktrees

This is a placeholder RULE.md file for the using-git-worktrees rule.

## When to Use

Use git worktrees to create isolated workspaces for feature development.

## How to Use

1. Create a worktree: `git worktree add .worktrees/branch-name -b branch-name`
2. Navigate to the worktree directory
3. Work in isolation from main
