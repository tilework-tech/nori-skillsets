Download and install a skillset package from the Nori package registrar.

Usage: /nori-registry-download <package-name>[@version]

Examples:
- /nori-registry-download my-skillset
- /nori-registry-download my-skillset@1.0.0

This command downloads the specified skillset package and extracts it to your profiles directory.
